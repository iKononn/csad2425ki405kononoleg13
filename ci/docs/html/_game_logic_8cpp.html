Makes a move on the Tic-Tac-Toe board.

This function updates the game board with the specified player's move. It places the player's symbol ('X' or 'O') on the board at the position determined by the move number (1-9). If the cell is already occupied, the function returns false. Otherwise, it places the symbol and returns true.

Parameters
boardA reference to a 2D vector representing the game board.
moveThe move number (1-9) representing the position on the board where the player wants to place their symbol.
playerThe symbol of the player ('X' or 'O') making the move.
Returns
true if the move was successful (the cell was empty and the symbol was placed), false otherwise (if the cell was already occupied).
